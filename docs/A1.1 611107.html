4. En el siguiente bloque de cdigo se muestra una grfica para la poblacin femenina. Para obtener esta parte de la poblacin, se aplica la condicin == "Female" sobre la columna Sexo de los datos. Es decir: data.Sexo == "Female"

Posteriormente, para el eje Y se utiliza el nivel de obesidad que se seleccion en el punto anterior, mientras que en el eje X se utiliza el peso, una de las variables cuantitativas de la base de datos. Como slamente me interesa la poblacin femenina, es necesario utilizar el cdigo descrito anteriormente sobre las dos columnas que se van a utilizar (nivel de obesidad y peso).

Por ltimo, se utiliz la librera plt para graficar. La funcin plt.figure ayuda a definir el tamao del grfico, y la funcin plt.scatter grafica los datos indicados.

In[3]:
%matplotlib inline

import matplotlib.pyplot as plt

# Definir "x" y "y", teniendo en cuenta slo a la poblacin FEMENINA
data_x = data.Peso[data.Sexo == "Female"]
data_y = data.NivelDeObesidad[data.Sexo == "Female"]

# Graficar datos
plt.figure(figsize=(10, 4))
plt.scatter(x = data_x, y = data_y, alpha = 0.05)
plt.show()
