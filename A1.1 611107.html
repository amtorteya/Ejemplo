2. Otra variable que hubiera decidido medir si formara parte del grupo de investigación sería:

CIRCUNFERENCIA DE LA CINTURA, la cual es una variable CUANTITATIVA

Ya que, investigando, este factor también se considera como indicador para detectar la obesidad. De acuerdo con la Secretaría de Salud de México (2016), "el riesgo aumenta si la circunferencia de cintura mide más de 80 centímetros en mujeres y más de 90 centímetros en el caso de los hombres". Por lo tanto, parece interesante ver la relación entre este factor y los demás presentados en la base de datos.

Otra variable podría ser:

HORAS A LA SEMANA DEDICADAS A HACER EJERCICIO, la cual es una variable CUANTITATIVA que podría indicar si la actividad física tiene algún impacto en el nivel de obesidad.

4. En el siguiente bloque de código se muestra una gráfica para la población femenina. Para obtener esta parte de la población, se aplica la condición == "Female" sobre la columna Sexo de los datos. Es decir: data.Sexo == "Female"

Posteriormente, para el eje Y se utiliza el nivel de obesidad que se seleccionó en el punto anterior, mientras que en el eje X se utiliza el peso, una de las variables cuantitativas de la base de datos. Como sólamente me interesa la población femenina, es necesario utilizar el código descrito anteriormente sobre las dos columnas que se van a utilizar (nivel de obesidad y peso).

Por último, se utilizó la librería plt para graficar. La función plt.figure ayuda a definir el tamaño del gráfico, y la función plt.scatter grafica los datos indicados.

In [3]:
%matplotlib inline

import matplotlib.pyplot as plt

# Definir "x" y "y", teniendo en cuenta sólo a la población FEMENINA
data_x = data.Peso[data.Sexo == "Female"]
data_y = data.NivelDeObesidad[data.Sexo == "Female"]

# Graficar datos
plt.figure(figsize=(10, 4))
plt.scatter(x = data_x, y = data_y, alpha = 0.05)
plt.show()
